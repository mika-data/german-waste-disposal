# German Waste Disposal (Mülltrennung)

This guide provides an overview of the waste separation system in Germany.

## Categories

- **Bioabfall (Brown/Green Bin):** Organic waste (fruit/vegetable scraps, garden waste, coffee grounds).
- **Altpapier (Blue Bin):** Paper and cardboard.
- **Gelbe Tonne/Gelber Sack (Yellow Bin/Bag):** Lightweight packaging (plastic, metal, composite materials).
- **Restmüll (Black/Grey Bin):** Non-recyclable residual waste (diapers, hygiene products, ashes, broken porcelain).
- **Glas (White/Green/Brown Containers):** Glass bottles and jars separated by color.

## Rules
1. Rinse containers briefly (don't need to be perfectly clean).
2. Do not stack plastic items inside each other (prevents sorting machines from recognizing them).
3. Hazardous waste (batteries, electronics, chemicals) must be taken to a Wertstoffhof.

## Local Variations
Disposal rules vary by municipality (Stadt/Landkreis). Always check your local Abfallkalender.
